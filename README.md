# List-KR Flat(ten)

Brave Browser와 같은 `!#include` 구문을 지원하지 않는 Ad-Blocker에 사용하기 위해 uBlock Origin 필터를 하나의 파일로 만드는 Repository입니다.

## 알아둘 사항

* 본 필터는 List-KR를 사용하는 **비공식** 필터입니다.
* 본 필터를 사용하면서 생기는 문제를 공식 저장소의 Issue로 등록하시면 **안됩니다**.
